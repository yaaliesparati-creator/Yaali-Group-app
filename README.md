App de gestión para Yaali Group — Encuentra Tu Lugar.

- 🏠 Yaali Inmobiliaria
- 🎨 Yaali Studio
- 📋 HYG Asesorías Mejoravit

## Cómo publicar gratis en Vercel

1. Sube esta carpeta a un repositorio en github.com
2. Ve a vercel.com y conecta tu cuenta de GitHub
3. Selecciona el repositorio y haz clic en Deploy
4. ¡Listo! Tu app estará en línea en ~2 minutos
